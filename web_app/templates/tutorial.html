Getting Started

Welcome to RNASEQ-MINI! This tutorial will guide you through setting up and running your first RNA-seq analysis.

Step 1: Installation

First, make sure you have RNASEQ-MINI installed:

git clone <repository-url>
cd rnaseq-mini
make setup

This will create the necessary conda environments and install dependencies.

Tip

Use make env-health to verify your installation after setup.

Setup Wizard

The interactive setup wizard helps you configure your analysis with intelligent defaults.

Step 2: Choose Your Workflow

The wizard will ask you to select your analysis type:

  • Standard RNA-seq: Differential expression analysis
  • Single-cell RNA-seq: 10x Genomics or similar data
  • Multi-omics: Integration with ATAC-seq, proteomics, etc.
  • Custom: Advanced or specialized analyses

Run the wizard with:

python scripts/setup_wizard.py

Analysis Workflows

Learn about the different analysis workflows and when to use each one.

Standard Workflow

Best for:

  • Bulk RNA-seq experiments
  • Differential expression analysis
  • Pathway enrichment studies

Tools: FastQC, Salmon, DESeq2, fgsea

Single-Cell Workflow

Best for:

  • 10x Genomics data
  • Cell type identification
  • Trajectory analysis

Tools: CellRanger, Scanpy, Seurat
